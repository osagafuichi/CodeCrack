# CodeCrack

**Debug and test any codebase, faster.**

CodeCrack is a language-agnostic IDE that reads your code, understands it, and
**automatically generates test cases that expose runtime and logical errors** —
so you spend less time writing tests and more time shipping.

It combines deterministic program analysis (parsing, static & flow analysis,
property-based testing) with an AI layer that explains *why* a test fails and how
to fix it. Your code runs in isolation, never loose on your machine.

## Language support

A shared, language-agnostic core plus per-language adapters. First-class targets:

- **Python**
- **JavaScript / TypeScript**
- **Java**

More to follow.
